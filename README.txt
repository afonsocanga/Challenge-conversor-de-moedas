Este projeto Java usa Maven para gerenciar dependências.

Para compilar e rodar:
1. Navegue até a pasta do projeto no terminal.
2. Compile com: mvn compile
3. Execute com: mvn exec:java -Dexec.mainClass="com.example.Main"

Certifique-se de que o Maven esteja instalado.

Dependência usada:
- Gson (para processar JSON da API)
